# Vietnamese Sentiment Analysis System

A Vietnamese NLP project for sentiment detection and order-intent extraction from social content.

## Key Features
- Vietnamese sentiment classification pipeline
- Order signal extraction from unstructured text
- Interactive dashboard for analysis and export
- Ready foundation for API-based deployment

## Tech Stack
- Data Processing: Pandas, text preprocessing
- Modeling: PyTorch, Transformers
- Collection: Selenium, BeautifulSoup
- Interface: Streamlit

## Methodology
- Collect and preprocess Vietnamese text data
- Run sentiment classification and entity extraction
- Evaluate outputs through a dashboard workflow
- Prepare the pipeline for future API integration

## Results
- Reduced manual effort in filtering customer comments
- Built a reusable pipeline for sentiment and order-intent analysis
- Created a practical NLP prototype suitable for portfolio presentation

## Installation
```bash
git clone https://github.com/laninh-tech/Vietnamese-Sentiment-Analysis-System.git
cd Vietnamese-Sentiment-Analysis-System
python -m venv .venv
pip install -r requirements.txt
streamlit run app.py
```
